ionic ---android开发一键式环境搭建


一、java环境配置

1、安装JDK开发环境
下载网站：http://www.oracle.com/


  • 开始安装JDK
    • 修改安装目录(也可以自定义目录)

      确定之后,单击“下一步”。
      注：当提示安装JRE时,可以选择不要安装。

2、配置环境变量：
对于Java程序开发而言,主要会使用JDK的两个命令：javac.exe、java.exe。路径：C:\Java\jdk 1.7.0 _09\bin。但是这些命令由于不属于windows自己的命令,所以要想使用,就需要进行路径配置。

右键单击“计算机-属性-高级系统设置”,单击“环境变量”。在“系统变量”栏下单击“新建”,创建新的系统环境变量。

接下来

(1)新建->变量名”JAVA_HOME”,变量值”C:\Java\jdk1.8.0_05”（即JDK的安装路径）
(2)编辑->变量名”Path”,在原变量值的最后面加上“;%JAVA_HOME%\bin;%JAVA_HOME%\jre\bin”
(3)在系统变量中新建->变量名“CLASSPATH”,变量值“.;%JAVA_HOME%\lib;%JAVA_HOME%\lib\dt.jar;%JAVA_HOME%\lib\tools.jar”

二、android环境（一键式安装）

1、 在一个磁盘目录下(我是在D盘)新建一个Android文件夹(文件夹的名字自己定义即可) ,打开此文件夹,在此文件夹下新建两个文件夹,一个是install,另一个是sdk。
2、 下载安装android-studio-bundle-141.2422023-windows.exe(根据自己的电脑操作系统下载)
下载地址:http://www.android-studio.org/index.php/88-download/

下载之后双击安装,弹出安装窗口后,点击Next,下一个窗口再点击Next,下一个窗口再点击 I Agree,此时的窗口会有两个路径要选择,第一个路径就是步骤1中创建的install文件夹路径,第二个路径就是步骤1中创建的sdk文件夹路径,然后点击Next,下一个窗口再点击Next,下一个窗口再点击Install,开始安装。

安装完成后,可以查看一下install和sdk这两个文件夹,里面就有了相应的文件。

3、配置环境变量

(1)新建->变量名”ANDROID_HOME”,变量值”D:\Android\sdk”（即SDK的安装路径,我的安装目录为D:\Android\sdk）
(2)编辑->变量名”Path”,在原变量值的最后面加上“%ANDROID_HOME%\platform-tools; %ANDROID_HOME%\tools; ” (windows10以下系统按照上面的写,windows10系统分两次添加)

配置完环境变量,打开命令窗口(window键+R,输入cmd之后按下回车),输入命令adb,下面会出现一些内容,此时表明配置完环境变量成功。

三、开始使用ionic

打开命令窗口(window键+R，输入cmd之后按下回车)，输入npm install -g ionic cordova
安装完成后，可以输入ionic -v查看ionic版本，输入cordova -v查看cordova版本

然后在一个磁盘里建一个ionic文件夹(我的建在E盘注意此文件夹路径不要有中文),打开文件夹,然后按下shift+鼠标右键，选择”在此处打开命令窗口”,进入命令窗口,输入ionic start myprj tabs创建一个App工程,
输入cd myprj进入工程目录,输入ionic serve启动应用,自动在浏览器打开,输入ionic platform add android为工程添加平台编译选项,输入ionic build android进行编译。
